## 浏览隐私是如何被“泄露”的？

在你打开网页的那一刻，你的设备其实就暴露了大量信息：

- **IP 地址**：显示你的位置、城市，甚至小区  
- **浏览器指纹**：操作系统、屏幕尺寸、插件等信息，组合起来几乎能定位你这个“唯一用户”  
- **Cookies 和缓存**：记录你的喜好、账号、购物车内容  
- **DNS 请求**：你访问过哪些网站，全都能被运营商看到  

这些数据一旦被广告公司、第三方平台、甚至不法分子获取，就能组成一份精准的用户画像，用于广告投放、账号定位，甚至诈骗。
